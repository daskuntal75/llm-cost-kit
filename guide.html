The 7-layer hierarchy

The most important concept in this kit. Where you put your instructions matters as much as what they say.

Layers ordered by cost-per-turn — top is most expensive (reloads every turn), bottom is cheapest (lazy-loaded).
LayerNameWorkspaceLoads
L1Cowork projectCoworkEvery turn
L2Cowork globalCoworkOnce / session
L3Code CLAUDE.mdCodeOnce / session
L4User PreferencesAllOnce / session — most efficient
L5MemoryChat + CoworkAuto-injected
L6SkillsAll ClaudeLazy / on trigger
L7Chat projectChatOnce / session

Quick start (Claude — 10 minutes)

# 1. Unzip and run setup
unzip claude-cost-kit.zip
cd claude-cost-kit
bash setup.sh
source ~/.zshrc

# 2. Follow the manual UI steps the script lists at the end:
#    L4 → Settings → Profile → Preferences → paste OUTPUT_RULES.md
#    L2 → Cowork → Settings → Global Instructions → paste cowork-global-instructions.md
#    L1 → For each Cowork project → paste tailored cowork-project-instructions.md
#    L7 → For each Chat project → paste tailored chat-project-instructions.md
#    L6 → Cowork → Customize → Skills → install 3 .skill zips from .build/

Model + effort routing

Claude Opus 4.7 era. Always-on, applies to every response.

TaskModelEffortMax output
Quick lookup, single functionHaiku 4.5Low360
Chat, summaries, draftsSonnet 4.6Medium960
Architecture / design reviewOpus 4.7High720
Routine feature codeOpus 4.7High1440
Security audit, complex refactorOpus 4.7xHigh1440

Escalation ladder: Low → Medium → High → xHigh → STOP. Never use Max — diminishing returns vs xHigh don't justify the spend. Escalate one step only after 2 failures at current tier.

Hourly cost pipeline

An optional LaunchAgent (macOS) refreshes your cost state and pushes live tally to all three machine-reachable instruction layers every hour:

# Wire the hourly pipeline (macOS LaunchAgent)
cp platforms/claude/cumulative/cumulative-cost-launchagent.sh ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/cumulative-cost-launchagent.plist

# What it does every hour:
update-claude-cost --code              # refresh ccusage session data
update-claude-cost --pull-api-spend    # pull API pool state via Admin API
update-claude-cost --emit-l7           # push cost tally to Chat project instructions (L7)
update-claude-cost --emit-l2           # push cost tally to Cowork global instructions (L2)
update-claude-cost --emit-l3-global    # push cost tally to Code CLAUDE.md (L3)

Scripts are self-contained in platforms/claude/scripts/. See platforms/claude/scripts/update-claude-cost for all flags.
